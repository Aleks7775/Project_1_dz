# Виджет банковских операций

## Описание

Это консольное приложение на Python для банковских операций, которое маскирует номер карты и 
счёта а так - же сортирует банковские операции по дате.

## Установка
Клонируйте репозиторий:
```
git clone https://github.com/username/project-x.git
```

## Использование:
1. Откройте виджет в вашем веб-браузере.
2. Введите номер карты или счета

## Пример:
*форма ввода*

Счет 73654108430135874305

Visa Platinum 8990922113665229

*форма вывода*

Счет **4305

Visa Platinum  8990 92** **** 5229

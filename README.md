# Виджет банковских операций

## Описание

Это консольное приложение на Python для банковских операций, которое маскирует номер карты и 
счёта а так - же сортирует банковские операции по дате.

## Установка
1. Клонируйте репозиторий:
```
git clone https://github.com/username/project-x.git
```
2. Перейдите в директорию проекта:
```
cd ваш_проект
```
3. Установите необходимые зависимости:
```
pip install -r requirements.txt

poetry add --group dev pytest
```


## Использование:
1. Откройте виджет в вашем веб-браузере.
2. Введите номер карты или счета

## Пример:
*форма ввода*

Счет 73654108430135874305

Visa Platinum 8990922113665229

*форма вывода*

Счет **4305

Visa Platinum  8990 92** **** 5229

## Тестирование 
Проект покрыт тестами. Для их запуска выполните команду ```poetry run pytest --cov```

## Функция генератор
Модуль является инструментом для эффективной работы с большими объемами данных транзакций,
используя возможности Python для обработки данных через генераторы. Эти генераторы должны 
позволять финансовым аналитикам быстро и удобно находить нужную информацию о транзакциях и проводить анализ данных.

## Пример:
### Генерация новых банковских карт
0000 0000 0000 0001, 0000 0000 0000 0002, 0000 0000 0000 0003
### Описание транзакции по запросу
Перевод организации, Перевод со счета на счет, Перевод с карты на карту
